# Projeto de Análise e Limpeza de Dados - Brasil 2022

Este projeto foi desenvolvido no Google Colaboratory com o objetivo de realizar a leitura, análise, limpeza e visualização de dados em formato CSV.

## Tema

Análise de indicadores socioeconômicos dos municípios brasileiros no ano de 2022.

## Objetivo

O objetivo do projeto é compreender a estrutura dos dados, identificar inconsistências, realizar a depuração da base e gerar um novo arquivo CSV limpo para futuras análises.

## Tecnologias utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab
- GitHub

## Etapas realizadas

1. Leitura do arquivo CSV;
2. Exploração inicial da base;
3. Identificação de valores nulos;
4. Verificação de registros duplicados;
5. Limpeza e padronização dos dados;
6. Geração de gráficos;
7. Exportação de um novo arquivo CSV limpo.

## Arquivo gerado

O arquivo tratado está disponível neste repositório:

`brasil_2022_limpo.csv`

## Conclusão

Após a análise e limpeza dos dados, foi possível gerar uma base mais organizada e adequada para utilização em futuras etapas do projeto.
